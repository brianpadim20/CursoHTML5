Aplicando ID al titulo

Para aplicar el atributo ID, en este caso el titulo, se utiliza el símbolo # + titulo del id por ejemplo #titulo-principal, el cual se usa en este ejemplo.

Lo cual es diferente a una clase, una clase (que empieza por un punto) se puede usar varias veces en el código, mientras que un id solo le pertenece a un elemento. En toda la pagina no debe haber otro elemento con ese identificador.

El concepto de ID también sirve para aplicar bookmarks o marcadores en HTML.
